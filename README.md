Student Management System

Overview

The Student Management System is a console-based application developed in C that uses file handling to manage student records. It allows users to add, display, search, update, and delete student information efficiently.

Features

- Add new student records
- Display all student records
- Search student by roll number
- Update student details
- Delete student records
- File-based data storage

Technologies Used

- C Programming
- File Handling
- Structures
- Standard C Libraries

Project Structure

Student-Management-System/
├── student.c
├── students.txt
└── README.md

How to Run

1. Compile the program:
   gcc student.c -o student
2. Run the executable:
   ./student
   (On Windows: "student.exe")

Menu Options

1. Add Student
2. Display Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit
